# 🌟 رؤية منتج: Autism Smart Bracelet

## 🎯 الهدف
مساعدة المصابين بالتوحد عبر جهاز ذكي يتعرف على إشارات التوتر العصبي المبكر، لتحسين التدخل العلاجي وتقليل السلوكيات الانفجارية.

## 🧠 المشاكل التي يعالجها
- صعوبة تعرف الأهل والمعالجين على إشارات التوتر المبكرة.
- تأخر التدخل مما يؤدي لانفجارات سلوكية خطيرة.

## 🌍 الأثر الإيجابي
- تقليل النوبات السلوكية.
- تحسين جودة حياة الطفل والأهل.
- استخدام البيانات لتحسين العلاج السلوكي مستقبلا.
